Тестовое задание, нужно развернуть чистый laravel, версия 9, базу можно использовать любую реляционную, и реализовать решение для чат бота в телеграмме, который при подключении к нему и запуска первичного скрипта (start, можно кнопку сделать, на усмотрение), выдавал бы в респонсе, "Имя подключившегося, хочу у вас работать!". В базу данных нужно реализовать 2 таблицы(подключившийся и таблицу логов с полями id, user_id и время подключения). И нужен будет код реализации, а так же запущенная версия для проверки (можно использовать стационарный компьютер) только для проверки. По времени реализации до вечера того же дня, но не менее 3 часов, когда получено тестовое задание.

Задумка такая. Каждый раз когда приходит сообщение должен дёргаться по api метод handle
Каждый раз он будет отвечать на последнее пришедшее сообщение.
То есть алгоритм
1. Пришло сообщение
2. Проверили пользака, есть у нас он или нет
3. Ответили ему на сообщение
